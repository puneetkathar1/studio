# **App Name**: Predictive Insights Pro

## Core Features:

- Market Data Ingestion: Ingest real-time market data (prices, volume, liquidity) from prediction market platforms like Polymarket and Kalshi.
- Market Microstructure Feature Computation: Compute market microstructure features to understand market dynamics and identify patterns.
- Fair Probability Envelope Builder: Build a Fair Probability Envelope (range, not point estimate) to determine a likely price range.  LLM tool incorporates features that it selects.
- Actionable Signal Generation: Generate actionable signals (BET/NO_BET/WAIT) based on the Fair Probability Envelope and other market data.  LLM tool incorporates features that it selects.
- Performance Ledger: Maintain a public, tamper-resistant performance ledger to track the accuracy of generated signals.
- Pro Feed: Deliver a premium feed with enhanced data and signals for paying subscribers.
- Daily Signal Brief: Provide a daily summary of key signals and market insights.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and intelligence, avoiding cliches related to money.
- Background color: Light grey (#F0F2F5), a desaturated version of the primary, to create a professional, data-focused environment in a light color scheme.
- Accent color: Yellow-gold (#FFC107), an analogous color, for highlighting actionable signals and key data points.
- Headline font: 'Geist Sans', sans-serif, for a computerized, techy look.
- Body font: 'Geist Mono', sans-serif, to pair with Space Grotesk for longer text.
- Use minimalist icons to represent market trends and trading signals.
- Use the esthetic of Shadcn UI for components
- Subtle animations to highlight important changes in market data.
